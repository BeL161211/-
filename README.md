# Название проекта

Получаем праздничные дни

## Как установить



Python3 должен быть уже установлен. 
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```

## Как Запустить проекта

Чтобы запустить проект надо в командной строке дойти до папки "Получаем праздничные дни" при помощи cd и вписать:
```
python main.py
```
